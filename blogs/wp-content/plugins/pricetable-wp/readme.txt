== Description ==

Pricetable WP plugin displays your pricetable on any WordPress post & page. It's created by using the latest CSS3 techniques .

I'm offering free support as I build this plugin up to version 1.0. 
Please use WordPress support forum to ask any query regarding any issue.

1. Front End - This is what the price table looks like .

2. Back End - The easy-to-use price table builder .

== Installation ==

1. Upload the entire `coming-soon-countdown` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Coming Soon Countdown menu to customize this plugin.
4. Finally Enjoy it.
